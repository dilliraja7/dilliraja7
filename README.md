## 🚀 Featured Projects

### 🧠 **NeuroPredict-AI: Early Alzheimer's Detection & Stage Prediction**
> **An end-to-end Deep Learning system utilizing hybrid neural network architectures for early-stage Alzheimer's classification.**
>
> 🔧 **Tech Stack:** `Python` • `MATLAB` • `CNN (Convolutional Neural Networks)` • `LSTM (Long Short-Term Memory)`

- 📈 **State-of-the-Art Performance:** Reached a classification accuracy of **99%** on PET/MRI clinical brain images.
- 🔬 **Hybrid Neural Architecture:** Combined spatial feature extraction of CNNs with chronological sequence mapping of LSTMs for precise disease progression modeling.
- ⚙️ **Advanced Pre-processing:** Programmed custom medical image pipelines, including noise reduction, skull-stripping, and dataset normalization.
- 📊 **Empirical Validation:** Verified model reliability via ROC (Receiver Operating Characteristic) curve evaluation and confidence metrics.
